# StudyGuy 3D — Project Brief
> Read at session start alongside 01_CURRENT_STATE.md

## Identity
- **Product:** StudyGuy 3D
- **Purpose:** Same as studyguy.net — AI study guide generator — but with a Three.js 3D landing page for higher visual impact and user acquisition
- **Repo:** https://github.com/ConnectSiteForge/studyguy3d (private)
- **Supabase project:** Same as studyguy.net (shared backend — https://zqsbhfaalhcnemahhdpq.supabase.co)
- **Status:** In development (started 2026-04-19)

---

## Relationship to studyguy.net
- studyguy3d is a **frontend variant only** — same backend (Supabase, Edge Functions, Stripe)
- All auth, DB, AI pipeline, and Edge Functions are shared with studyguy.net
- Never modify studyforge/studyguy.net Cloudflare Pages, repo, or Supabase config
- The 3D landing page (index.html) is the main differentiator

## Three.js Scene (index.html)
- Full-screen canvas behind all content (`z-index: 0`)
- Particle field: 1800 points, slow upward drift
- Geometric objects: IcosahedronGeometry, OctahedronGeometry, TetrahedronGeometry — wireframe + solid, slowly rotating
- Connection lines between nearby particles
- Mouse parallax: camera follows mouse
- Scroll parallax: camera shifts on scroll
- Color: #00c896 (electric) to match design system

## Config / Build
